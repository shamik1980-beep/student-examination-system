# Student Examination System - Java MVC

This is a simple Java MVC application for managing student examination details.

## Features
- Add student
- Search student by ID
- Update marks
- Display all students
- Pass/Fail result generation

## Project Structure
src/com/exam/
- Main.java
- model/Student.java
- view/StudentView.java
- controller/StudentController.java

## Branches
- main
- feature/add-student
- feature/search-student
- feature/update-marks
